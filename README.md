# uno-py
Recreating UNO on the CLI with Object Oriented Python (OOP)

# Usage
Insert your nickname
Play card by inputting its index or Buy inputting 'b' or 'buy'
Cards |?!| (Color Change) and |+4| (Plus 4) give you the chance to change colorof table card
Game ends at the first player to empty his/hers deck
